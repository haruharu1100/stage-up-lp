### 手順（営業優先シート）

1. スプレッドシートを開く
2. **2行目以降の全範囲をクリア**

   - `Delete` キーで全消去
3. **L列・M列を完全空白化**
   - 列ヘッダの `L` をクリック → `Shift+クリック`で `M` → 範囲選択
   - `Delete` キーで全消去（管理番号1〜25行の「未送付/未実施」もこれで消える）
4. **折り返し設定を「切り詰め」に**
   - A〜P 列を選択 → 表示形式 → 折り返し → **切り詰め**
5. **TSV を貼り付け**
   - ターミナルで `open _sheets/2026-05-30_ALL_UNIFIED.tsv`
   - 「営業優先シート（全132社・統合版）」セクションの **132行**（先頭ヘッダ7行を除外）を選択 → コピー
   - スプレッドシートの `A2` をクリック → **Cmd+Shift+V**（プレーンテキスト貼り付け）

### 手順（タスク管理シート）

1. タスク管理シートタブに切替
2. 同じく A2 以降を全クリア、L:M 列クリア、折り返し「切り詰め」
3. TSV ファイル内の「タスク管理シート（全132社）」セクションの **132行** を選択 → コピー
4. A2 に Cmd+Shift+V で貼り付け

### 補完作業（管理番号 1〜51 のみ）

統合TSVでは管理番号 1〜51 の **D列（法人番号）・E列（住所）・F列（設立日）** に「（要確認）」と入っています。

既存スプレッドシートの該当セル値をコピーするか、再調査して埋めてください。

## 1. 貼り付け前の準備（スプレッドシート側）

|---|---|---|

設定方法：列ヘッダ（A B C…）を右クリック → 「列のサイズを変更」→ ピクセル指定

### 折り返し設定（最重要）

1. 貼り付け先の範囲（または A〜P 全列）を選択
2. **表示形式 → 折り返し → 「切り詰め」** を選択
   （ショートカット：選択中に `表示形式` メニュー → `折り返し` → `切り詰め`）
3. これで長文DM文章があっても 1 行表示・行高 21px のまま保たれます

### 行高プリセット

1. 行ヘッダ（1 2 3…）を全選択
2. 右クリック → 「行のサイズを変更」→ **21px** を指定

## 2. 貼り付け手順

1. ターミナルで TSV ファイルを開く：
   ```
   open _sheets/2026-05-30_osaka-62.txt
   ```
2. テキストエディタで「営業優先シート（62社…）」セクションの 62 行（タイトル行除く）を選択 → コピー
3. スプレッドシートの **A列の最終行直下** をクリック
4. **Cmd+Shift+V**（プレーンテキスト貼り付け）
   - ❌ Cmd+V だと書式が付着して崩れます
   - ✅ Cmd+Shift+V でテキストのみ貼り付け
5. タスク管理シートにも同じ要領で貼り付け（62 社ぶん）

## 5. データ仕様（参考）

- 列構成：16 列（A〜P）
- L列・M列：**完全空白固定**（運営側で手入力する欄）
- DM文章 (O列)：1 セル 1 行・改行コード/`\n` 禁止・区切りは全角スラッシュ ` ／ `
- TAB 区切り（タブで列が決まる）
- 文字コード：UTF-8

## 6. AI出力ルール（再発防止）

AI が今後 TSV を生成する際は以下を遵守：

1. DM文章中に `\n` リテラルを入れない（区切りは ` ／ `）
2. DM文章中に実改行コード `\x0A` を入れない
3. L列・M列は空のタブ 2 つ（`\t\t`）で出力
4. 出力後に必ず以下のチェックを実施：
   ```bash
   grep -c '\\n' _sheets/該当ファイル.txt   # → 0 件であること
   awk -F'\t' 'NF!=16 {print NR": "NF" cols"}' _sheets/該当ファイル.txt   # → データ行は16
   ```
