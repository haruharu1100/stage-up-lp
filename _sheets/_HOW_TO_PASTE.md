# 営業優先シート／タスク管理シート 貼り付け手順（必読）

⚠️ **このファイルを読まずに貼り付けると表示が崩れます。必ず手順通りに実施してください。**

---

## 1. 貼り付け前の準備（スプレッドシート側）

### 列幅プリセット（推奨）

| 列 | 項目 | 列幅(px) |
|---|---|---|
| A | 取得日 | 90 |
| B | 管理番号 | 60 |
| C | 会社名 | 220 |
| D | 法人番号 | 120 |
| E | 住所 | 280 |
| F | 設立日 | 90 |
| G | HP URL | 100 |
| H | HP有無 | 60 |
| I | LINE有無 | 60 |
| J | スマホ対応 | 70 |
| K | 業種予測 | 140 |
| **L** | **DM送付（空白）** | **80** |
| **M** | **面談状況（空白）** | **80** |
| N | 備考 | 240 |
| O | **DM文章** | **320**（広げすぎ厳禁） |
| P | LP URL | 280 |

設定方法：列ヘッダ（A B C…）を右クリック → 「列のサイズを変更」→ ピクセル指定

### 折り返し設定（最重要）

1. 貼り付け先の範囲（または A〜P 全列）を選択
2. **表示形式 → 折り返し → 「切り詰め」** を選択
   （ショートカット：選択中に `表示形式` メニュー → `折り返し` → `切り詰め`）
3. これで長文DM文章があっても 1 行表示・行高 21px のまま保たれます

### 行高プリセット

1. 行ヘッダ（1 2 3…）を全選択
2. 右クリック → 「行のサイズを変更」→ **21px** を指定

---

## 2. 貼り付け手順

1. ターミナルで TSV ファイルを開く：
   ```
   open _sheets/2026-05-30_osaka-62.txt
   ```
2. テキストエディタで「営業優先シート（62社…）」セクションの 62 行（タイトル行除く）を選択 → コピー
3. スプレッドシートの **A列の最終行直下** をクリック
4. **Cmd+Shift+V**（プレーンテキスト貼り付け）
   - ❌ Cmd+V だと書式が付着して崩れます
   - ✅ Cmd+Shift+V でテキストのみ貼り付け
5. タスク管理シートにも同じ要領で貼り付け（62 社ぶん）

---

## 3. 貼り付け後チェックリスト（毎回必須）

- [ ] **L列・M列が完全空白か**（「未送付」「未実施」が混入していないか）
- [ ] **行の高さが 21〜30px に収まっているか**
- [ ] **DM文章列 (O) の幅が 320px 程度に保たれているか**
- [ ] **A〜P 列まですべて画面で見えるか**
- [ ] **タスク管理シートには LP URL がある会社（62 社）のみ入っているか**
- [ ] DM文章セル内に `\n` の 2 文字（バックスラッシュ＋n）が残っていないか
  - 残っていたら：そのセルをコピー → `Cmd+F` → `Cmd+Shift+H`（検索と置換）→ 検索文字列 `\n` → 置換 ` ／ ` → すべて置換

---

## 4. 表示が崩れたときの応急処置

1. 崩れた範囲を選択
2. **表示形式 → 折り返し → 切り詰め**
3. 行ヘッダ全選択 → 右クリック → 行高 21px に再設定
4. それでも崩れる場合：O列（DM文章）の列幅を 320px に固定

---

## 5. データ仕様（参考）

- 列構成：16 列（A〜P）
- L列・M列：**完全空白固定**（運営側で手入力する欄）
- DM文章 (O列)：1 セル 1 行・改行コード/`\n` 禁止・区切りは全角スラッシュ ` ／ `
- TAB 区切り（タブで列が決まる）
- 文字コード：UTF-8

---

## 6. AI出力ルール（再発防止）

AI が今後 TSV を生成する際は以下を遵守：

1. DM文章中に `\n` リテラルを入れない（区切りは ` ／ `）
2. DM文章中に実改行コード `\x0A` を入れない
3. L列・M列は空のタブ 2 つ（`\t\t`）で出力
4. 出力後に必ず以下のチェックを実施：
   ```bash
   grep -c '\\n' _sheets/該当ファイル.txt   # → 0 件であること
   awk -F'\t' 'NF!=16 {print NR": "NF" cols"}' _sheets/該当ファイル.txt   # → データ行は16
   ```
