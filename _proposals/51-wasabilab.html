about us

// クリエイティブとテクノロジーの境界線で、新しい解を実験し続けるラボです。

株式会社ワサビラボは、奈良発のクリエイティブ・スタジオです。私たちは「答えありき」のクリエイティブを作りません。お客様の課題と向き合い、毎回ゼロから「新しい解」を実験するスタンスを大切にしています。

「ワサビ」という名前は、ピリッと効いた刺激と、本物の質感の象徴です。派手なだけのアウトプットではなく、本当に効くアイデアと、確かな実装力を兼ね備える存在を目指しています。

ブランド、デザイン、コード、メディア。あらゆる手段を使って、お客様のビジネスに新しい味付けを。

Brand Lab.

ブランドの核を見つけ出し、ビジュアルとストーリーで形にする。新規ブランド立ち上げから、既存ブランドの再定義まで対応します。

stack: brand strategy / identity / visual / storytelling

// lab.02

Code Lab.

Web・モバイル・プロトタイプまで、コードで価値を実装するチーム。最新フロントエンドから業務システムまで幅広く対応します。

stack: React / Next.js / Node / TypeScript / Mobile

// lab.03

Media Lab.

stack: photo / video / motion / editorial / social

// 私たちのアウトプットがピリッとする、4つの実験的アプローチです。

// exp_0101

Hypothesis-first creativity.

毎プロジェクト、必ず「仮説」から始めます。なぜこのアウトプットが効くのか、検証可能な仮説を立てた上で制作に入ることで、感覚論ではないクリエイティブを実現します。

// method
Hypothesis → Prototype → Test → Refine

// exp_0202

Cross-functional team.

ブランド戦略・デザイン・コード・メディアまで横断する小さなチームが、一つの課題に挑みます。分業ではなく統合の力で、一貫したアウトプットを実現します。

// crew
Strategist · Designer · Engineer · Editor

// exp_0303

Rapid prototyping.

机上のプレゼンではなく、すぐに動くプロトタイプを見せながら議論を進めるスタイル。意思決定のスピードと精度を同時に上げます。

// tools
Figma · Framer · Notion · Code

// exp_0404

Small but bold.

大きな代理店にはできない、小さなチームならではの実験的なアウトプットを大切にしています。「無難な選択」を意識的に避け、印象に残るものを作る。

// belief
Bold over safe / Sharp over generic
