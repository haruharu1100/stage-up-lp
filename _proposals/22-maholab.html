our mission

We build infrastructure
for the next generation.

テクノロジーは、生活の風景を変える道具です。私たちが向き合うのは、最先端の研究ではなく、現場で「実際に動くもの」を設計し、社会の隅々まで届けること。

私たちは、技術と現場の橋渡し役として、データの可能性を社会のインフラに変換していきます。

研究室で生まれた技術が、現場で使われずに眠ったままになることに、私たちはずっと違和感を持ってきました。だからこそ「現場で本当に役立つ実装」だけを追求し、専門領域に深く入り込み、共に成果を作る姿勢を貫きます。

technology stack

私たちの研究・開発の中核となる3つの技術領域です。

// AI / Machine Learning

機械学習・予測モデル

需要予測・異常検知・画像認識・自然言語処理まで、産業現場のデータから価値を引き出すモデル設計を行います。

PyTorch · TensorFlow · scikit-learn · YOLO · BERT

// IoT / Edge

IoT・エッジコンピューティング

Raspberry Pi · NVIDIA Jetson · LoRaWAN · MQTT · AWS IoT

// Data Platform

データプラットフォーム構築

企業内に散在するデータを統合・可視化・分析できる基盤を構築。意思決定の質とスピードを引き上げます。

Snowflake · BigQuery · dbt · Looker · Apache Airflow
