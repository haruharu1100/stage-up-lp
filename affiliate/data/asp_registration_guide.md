# ASP登録ガイド（通信ジャンル特化）

## 登録優先順位と所要時間

### 1. A8.net（最優先・即日承認）

- URL: https://www.a8.net/
- 必要時間: 30分（口座情報・サイト情報入力のみ）
- 審査: ほぼ即承認（サイトなしでも登録可。「ファンブログ」無料利用可）
- 通信案件:
  - 楽天モバイル: 報酬 6,000-8,000円/件
  - 楽天Turbo: 報酬 10,000-13,000円/件
  - mineo, IIJmio, BIGLOBE: 報酬 3,000-15,000円/件
- 振込: 月末締め・翌々月15日振込（最低1,000円から）

**登録時のサイト情報入力**:
- サイト名: 「通信比較ナビ」等
- サイトURL: Vercelデプロイ後のドメイン
- カテゴリ: 「通信・PCインターネット」を選択

### 2. afb（アフィビー）（1-2日承認）

- URL: https://www.afi-b.com/
- 必要時間: 60分（本人確認書類アップロード必要）
- 審査: 1-2営業日
- 通信案件強み:
  - SoftBank Air: 報酬 15,000-20,000円/件（A8より高単価）
  - GMOとくとくBB系: 報酬 12,000-18,000円/件
  - 法人向け通信: 報酬 30,000-50,000円/件
- 振込: 月末締め・翌月末振込（最低777円から）
- 特徴: 「消費税分上乗せ」振込で実質報酬+10%

### 3. バリューコマース（2-3日承認）

- URL: https://www.valuecommerce.ne.jp/
- 必要時間: 90分（サイト審査あり）
- 審査: 2-3営業日（サイトが必要、Vercel公開後に申請）
- 通信案件強み:
  - ドコモ光、auひかり: 大手キャリア独占案件
  - 報酬 18,000-25,000円/件
- 振込: 月末締め・翌々月15日振込（最低1,000円から）
- 特徴: Yahoo!プロモーション連携でCV計測が安定

### 4. felmat（フェルマ）（3-6ヶ月後・クローズドASP）

- URL: https://t.felmat.net/
- 加入方法: 紹介制 or 実績証明（月10万円以上の他ASP実績）
- 通信案件強み: 高単価独占案件多数（同案件で他ASP+30-50%）
- A8で3-6ヶ月運用後に申請推奨

### 5. レントラックス（最上位クローズド）

- URL: https://www.rentracks.jp/
- 加入方法: 紹介制（A8等の業界知人経由のみ）
- 通信案件: 報酬 30,000-60,000円/件のVIP案件
- 月50万円以上の安定報酬が見えてきてから挑戦

## 登録手順チェックリスト

### 事前準備
- [ ] 銀行口座（個人で可、楽天銀行推奨：振込手数料無料案件あり）
- [ ] 開業届（税務署提出、屋号があると印象◎）
- [ ] 身分証明書（運転免許 or マイナンバー）
- [ ] サイトURL（Vercelデプロイ後の本ドメイン or サブドメイン）
- [ ] サイト名・運営者名・運営者連絡先

### A8.net 登録
1. トップ右上「会員登録」→ メールアドレス入力
2. メール認証 → 本登録フォーム
3. サイト情報入力（複数サイト登録可）
4. 口座情報入力
5. 即時マイページ利用可能 → 通信ジャンルから案件提携申請

### afb 登録
1. 「アフィリエイト無料会員登録」→ 必要情報入力
2. 本人確認書類アップロード
3. サイト審査（1-2営業日）
4. 承認後、通信ジャンルから案件提携申請

### バリューコマース 登録
1. 「無料で会員登録」→ サイト情報入力
2. サイト審査（2-3営業日、サイトのコンテンツが必要）
3. 承認後、通信ジャンルから案件提携申請

## 案件提携申請のコツ

1. **複数申請OK**: 同じカテゴリで5-10社並行申請
2. **却下されたら3ヶ月待つ**: 同案件への再申請は時間を空ける
3. **サイト記述充実**: 「比較サイト運営」「個人ブロガー」より「メディア運営」と書く方が通りやすい
4. **NG案件チェック**: 商標KW入札禁止案件は事前に確認

## 報酬の流れ

```
広告出稿 → LPアクセス → アフィリリンク クリック
   ↓
ASP計測（クリックID付与）
   ↓
公式サイトで申込・契約
   ↓
ASPで「CV発生」記録（クッキー24h以内）
   ↓
広告主が承認/否認判定（1-2ヶ月）
   ↓
ASPで「確定報酬」となる
   ↓
締日翌月 or 翌々月に振込
```

**実際の現金化まで2-3ヶ月かかる**点に注意。資金繰り計画必須。
