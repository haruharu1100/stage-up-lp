# 通信アフィリLP 運用ランブック

このディレクトリは「通信ジャンル × Google/Yahoo検索広告 × アフィリエイト」の収益化システム。
B2B提案LP（`_proposals/`）と完全に別系統で動作。

## 目標 KPI

| 指標 | 目標値（3ヶ月後） |
|---|---|
| 月間売上（アフィリ報酬） | **100万円** |
| 月間広告費 | 40-50万円 |
| ROAS | **200%以上** |
| 月間CV | 77件（1日2.5件） |
| 平均CV率 | 2-3% |
| 平均CPA | 6,000-7,000円 |

## ディレクトリ構成

```
affiliate/
├── data/
│   ├── offers.json          # ASP案件マスター
│   ├── niches.json          # ニッチ×ペルソナマスター

│   └── tracking_config.json # GA4 / Clarity / ASP計測タグ
├── scripts/
│   └── build_aff_lps.py     # LP一括生成

└── README.md                # 本ファイル
```

## 公開URL設計

| パス | 内容 |

## 運用フロー（日次）

### 朝（30分）
1. **GA4 / Microsoft Clarity** で前日のセッション・直帰率を確認
2. **ASP管理画面** で前日CV確認、確定報酬チェック
3. **Google広告管理画面** で前日CPC・CPA・予算消化を確認
4. **判断**：
   - CPA > 報酬の70% のLP → **即停止**
   - CPA < 報酬の50% のLP → **予算3倍**
   - CV率 < 0.5% のLP → **3日連続なら停止**

### 昼（60分）
1. **Clarityヒートマップ** で勝ちLPの離脱ポイント分析
2. **FV・CTA・FAQ** のテキスト改善
3. 新ニッチKW追加：ラッコキーワード→Googleサジェスト

### 夕方（90分）
1. **新LP制作**：勝ちパターン横展開
2. **広告文ABテスト**：見出し3パターン追加投入
3. **競合LP分析**：上位3社のFV・CTAを観察

## 勝ち負け仕分けルール（厳格運用）

### 即停止ライン
- 100クリックでCV 0件
- CPA > 報酬額の80%（赤字確定）
- 広告の品質スコア3以下

### スケール判断ライン
- 直近7日のCV率 ≥ 2.0%
- 直近7日のROAS ≥ 180%
- → 日予算を2-3倍に引き上げ

### 横展開ライン
- 勝ちLP × 別案件（同カテゴリ）
- 勝ちLP × 類似ニッチ
- 勝ち広告文 × 類似KW群

## 景表法・広告ポリシー NGリスト

LP本文・広告文ともに**絶対に使わない**表現：

- 最速、最安、日本一、業界No.1、世界一、トップ
- 絶対、必ず、100%、完全、永久、保証
- 激安、超お得、爆速
- 「○○より安い」（比較対象明記なしの場合）
- 「全員」「みんな」（範囲明示なしの場合）

代替表現例：
- ❌「業界最安」→ ✅「コスパで選ぶ」
- ❌「絶対繋がる」→ ✅「広いエリアで利用可能」
- ❌「100%満足」→ ✅「ご利用者から高評価」
