## 変更詳細

### 1. remote_worker（テレワーク・Web会議落ちない回線）

**変更理由：** リサーチレポート推奨H-3「テレワーク系LPでスマホセット割の永年割引を前面訴求」。競合LPはCB金額よりも「永年でスマホが月〇〇円割引」を大きく打ち出している。auひかり（報酬25,000円）が主力案件。

#### pain_points 追加（2件）
```
【追加1】auひかりのスマホセット割（auスマートバリュー）は光回線申込後いつから適用されるのか・別途手続きが必要なのか・適用されると月いくら割引になるのか

【追加2】光回線のキャッシュバック（最大数万円）はいつ・どのように受け取るのか・手続きを忘れたり期限を過ぎた場合は受け取れなくなるのか
```

**CV期待効果：** セット割適用タイミングの不明確さと、CB受取を忘れるリスクは「よく調べてから申し込もう」という先送りの原因になる。申込前の典型的な疑問を先回りして解消することで申込ボタンクリックへの障壁を低減。

#### search_keywords 追加（3件）
```
auひかり スマートバリュー いつから 適用 申込後 別途 手続き 必要か 2026
テレワーク 光回線 セット割 永年 auひかり auスマホ 月いくら 割引 2026
光回線 キャッシュバック 受取 期限 忘れ 何ヶ月後 auひかり BIGLOBE光 2026
```

#### primary_appeal 冒頭変更

**変更前：**
```
Zoom・Teams映像が安定する上り速度重視の光回線・工事立ち合いは通常2〜3時間で...
```

**変更後：**
```
【auスマホ・UQモバイルユーザーは光回線を変えるだけでスマホ料金が永年月最大1,100円割引】auひかり（auスマートバリュー）はauスマホ・UQモバイルとのセット割で光回線申込後に月最大1,100円のスマホ割引が永年適用（セット割は開通確認後から自動適用・条件あり・詳細は申込時確認）・2年・3年と使い続けるほど割引累計が大きくなるため長期のテレワーク回線として実質コストが下がる・Zoom・Teams映像が安定する...（以下同）
```

**CV期待効果：** 「月1,100円×24ヶ月＝26,400円の永年節約」という長期的なお得感は、一度見た競合LPに戻らずそのまま申し込みに進む誘因になる。

### 2. one_person_room（一人暮らし即日WiFi）

**変更理由：** リサーチレポート推奨H-2「ファーストビュー直下に結論セクションを追加」。競合LPは「SoftBank Air が実質最安」のように簡潔な結論を先出しするが、自社LPは詳細比較が先で情報過多だった。

#### search_keywords 追加（1件）
```
SoftBank Air ZEUS WiFi どっち 結論 費用重視 CB重視 一人暮らし ホームルーター 2026
```

#### primary_appeal 冒頭変更

**変更前：**
```
【3年総費用CB差引後の実質比較2026】費用重視→ZEUS WiFi：3年実質約87,548円・月換算約2,432円相当（CB10,000円差引後・初期費用込み・条件あり）／CB最大化→SoftBank Air：3年実質約161,248円目安・...いずれも解約金なし・工事不要・翌日発送・
```

**変更後：**
```
【結論】一人暮らしのホームルーターは「費用重視→ZEUS WiFi（月2,618円〜・解約金0円・新規CB10,000円）」か「キャッシュバック重視→SoftBank Air（月5,368円・CB最大35,000円・8ヶ月後自動振込）」の2択・いずれも工事不要・申込翌日発送・縛りなし▼詳細比較↓・3年総費用CB差引後の実質比較2026：費用重視→ZEUS WiFi：3年実質約87,548円・...（以下同）
```

**CV期待効果：** スキャンユーザーが最初の2行で「自分はどちらを選ぶべきか」が判断できるため、情報過多による離脱を防止。比較疲れ防止効果で申込率向上が見込まれる。

### 3. electricity_one_person（一人暮らしの電気代節約）

**変更理由：** リサーチレポート推奨M-2「楽天でんきポイント特典条件の明確化」。一人暮らしで月電気代2,000〜3,000円程度の場合、特典対象外になるケースがあり、景表法上も条件明示が必要。

#### pain_points 追加（1件）
```
楽天でんきの新規申込ポイント特典（最大8,000pt相当）は一人暮らしでも対象になるのか・月いくら以上の電気代がないと特典が適用されないのか・電気代が月2,000〜3,000円程度しかない場合でも貰えるのか
```

**CV期待効果：** 「一人暮らしで電気代が少ないと特典が貰えない」という隠れた不安を事前に提示・解消することで、申込後の「騙された感」を防ぎ、ユーザーの信頼を高めて申込を促進。

#### primary_appeal 変更（楽天でんきポイント条件追記）

**変更前：**
```
楽天ポイントで電気代の一部を充当可能・切替後も...
```

**変更後：**
```
楽天ポイントで電気代の一部を充当可能（楽天でんき新規申込特典のポイント付与は2カ月合計6,000円以上の電気代が必要条件・一人暮らしで月電気代が2,000〜3,000円程度の場合は特典対象外の可能性あり・申込前に公式シミュレーターで個別確認を推奨）・切替後も...
```

**CV期待効果：** 景表法リスクの低減と同時に、「条件を確認してから申込む」というアクションに誘導できるため、シミュレーター経由の申込件数増加が期待される。
