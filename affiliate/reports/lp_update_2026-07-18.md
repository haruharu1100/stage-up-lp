**変更箇所: pain_points（新規追加1項目）**

| | 内容 |
|---|---|
| 変更前 | （この不安が未回答）|
| 変更後 | 「競合のMugen WiFi（月3,190円/100GB）には30日間のお試し返品保証があると聞いたがZEUS WiFiにはそういったお試し期間はないのか・出張先で電波が弱くて使えなかった場合に解約金・端末返送費用をかけずに辞められるのか」 |

| | 内容 |

### ニッチ②: gamer（ゲーマー向け低Ping回線）

**対応実リンク案件:** auhikari（¥25,000）、sonet-hikari（¥18,000）

**改善理由:**  
auhikariは報酬¥25,000で実リンク案件中の最高額。にもかかわらずpain_pointsが4項目と少なく、抽象的（「NAT設定はやり直しになるのか」等の答えが一行で解決できる質問のみ）。search_keywordsも8項目のみで、ゲーマー固有の検索意図（NAT・ポート開放・v6プラスとの比較・工事期間中の対策）が網羅できていなかった。

**変更箇所: pain_points（4項目→5項目・具体化）**

| 変更前 | 変更後 |
|---|---|
| 「光回線の開通工事期間（2〜4週間）中はゲームが一切できなくなるのか」 | 「光回線開通工事の2〜4週間の空白期間中、FPS・対戦ゲームはポケットWiFiで代替できるのか・Ping（遅延）が高くてオンライン対戦は実用的でないのか・工事期間中もゲームを続けるための現実的な方法はあるのか」 |
| 「独自回線に変えると今のルーター設定やポート開放はやり直しになるのか」 | 「auひかり（独自回線）に乗り換えると今使っているルーターのポート開放・NAT設定・DMZ設定はすべてやり直しになるのか・NATタイプがオープン（タイプA/B）になる保証はあるのか・設定手順はサポートで教えてもらえるのか」 |
| 「マンション住まいの場合、管理組合への許可申請が必要になるケースはあるのか」 | 「賃貸マンションでauひかりの工事申請をするとき管理組合・大家への許可申請が必要か・工事を断られた場合でも光コラボ（v6プラス対応）なら工事なしで切り替えられるのか」 |
| 「契約期間中に引越しが決まった場合の解約金はいくらになるのか」 | 「auひかりは最大36ヶ月の契約縛りがある場合があると聞いたが、引越し時の解約金はいくらか・エリア内なら工事なしで移転できるのか」 |
| （新規追加） | 「独自回線（auひかり）とフレッツ光コラボ（v6プラス対応）では夜20〜23時のFPS・対戦ゲーム中のPing値・パケットロスに実質的な差があるのか」 |

**変更箇所: search_keywords（8項目→13項目・追加5件）**

追加した5キーワード：
- 「光回線 工事 空白期間 ゲーム ポケットWiFi Ping 遅延 対戦 代替 できるか 2026」
- 「auひかり ポート開放 NAT タイプ 乗換後 やり直し 設定 方法 サポート 2026」
- 「auひかり 賃貸 マンション 工事 許可 管理会社 断られた 光コラボ 代替 2026」
- 「auひかり 解約金 転勤 引越し エリア外 縛り 36ヶ月 移転 費用 2026」
- 「光コラボ v6プラス auひかり Ping 比較 夜 混雑 FPS ゲーム 安定 2026」

**変更箇所: primary_appeal（NAT/ポート開放・工事期間中の対応策を追記）**

追記内容：ポート開放のサポート確認方法・工事期間中のポケットWiFiのPing特性・賃貸で独自回線工事不可時のフレッツ光コラボへの代替誘導を明示。

**期待効果:** NAT設定やポート開放への不安を具体的に解消 → auひかりへの申込の踏み止まりを減少。¥25,000案件へのCV誘導強化。

### ニッチ③: student_new（大学生・新入生向け4月需要）

**対応実リンク案件:** zeus-wifi（¥11,000）、rakuten-mobile（¥7,000）

**改善理由:**
