## 3. ギャップ分析

### 3-1. 競合が強化している訴求 vs 自社現状

| 競合の強化トレンド | 自社の現状 | ギャップ |
|------------------|-----------|---------|
| キャリア別おすすめ分類（au/ドコモ/SB/楽天） | 一部ニッチ（mnp_from_*）で対応済み | 一人暮らし・引越し等の主要ニッチでキャリア別訴求が薄い |
| 「2年サイクル乗り換えでCB繰り返し活用」戦略訴求 | moving_fiberで一部言及あり | 単発CBのみ訴求・繰り返し乗換戦略の提案が不足 |
| 「2026年7月補助金再開」ニュース性フック | electricity系ニッチで言及あり | 電気系LPでの訴求タイミングの即応性が問題（鮮度） |
| シミュレーション→申込の2ステップCV導線 | 現状のprimary_appealはテキスト情報提供型 | インタラクティブな試算導線が競合に比べ弱い |
| 「月額500円〜」「最安」系インパクト見出し | 景表法NG表現を避けているため使用不可 | 安さの訴求に景表法準拠で対抗できる表現の工夫が必要 |
| 倒産リスク・電力安定性の安心訴求（電気系） | 電気系ニッチのprimary_appealで言及済み | 競合に比べ訴求の前面への配置が不足している可能性 |
| 「後悔しない」「落とし穴」系ネガ訴求コンテンツ | 各ニッチのpain_pointsに落とし穴要素は含む | 見出し・CTAへの反映が不足している可能性 |
| povo「基本料0円」の特殊ポジション訴求 | second_lineニッチのprimary_appealで言及 | 副回線ニッチでpovoの0円継続条件（180日トッピング）の具体訴求を強化余地あり |

### 3-2. 競合に対して自社が優位な点

| 自社の強み | 根拠 |
|-----------|------|
| ペルソナ別の超細分化（24ニッチ） | 競合の多くは「一人暮らし全般」等の大括りで、ニッチが浅い |
| 痛点（pain_points）に基づいたQ&A型訴求 | 競合は価格比較中心。自社は「なぜ悩むか」から逆算した訴求 |
| 景表法準拠の安全な訴求設計 | 違反リスクのある競合が多数。長期的なサイト安全性で差別化 |
| 電力・ガス系ニッチへの展開 | WiFi専業競合が多く、電力・ガスへの展開で競合が少ないブルーオーシャン |
| キャッシュバック受取条件の明示 | 「条件あり」明記が消費者信頼を高め、中長期でCV率に寄与 |

## 4. 改善推奨事項

### 優先度：高

#### 4-1. 電気系LP「2026年7月補助金再開」訴求の即時更新（electricity_one_person・electricity_family・electricity_gas_set）
- **課題：** 競合の複数サイトが「2026年7月補助金再開見込み」をフックに切り替えを促している。自社電気系ニッチの `primary_appeal` にはすでに補助金言及があるが、LP上での鮮度・配置が競合に比べ劣る可能性がある。
- **対応：** 電気系LPの見出し・ファーストビューに「2026年7月〜補助金再開見込みのいま乗り換えると？」等のニュース性フックを配置。景表法NG表現を避けつつ、政府発表に基づく事実として訴求する。
- **期待効果：** 旬なニュース性でクリック率（CTR）向上・検索上位記事との差別化。

#### 4-2. 一人暮らし・引越し系LPへのキャリア別訴求追加（one_person_room・moving_urgent・student_new）
- **課題：** 競合上位サイトで「auユーザーはBroad WiMAX」「ドコモユーザーはhome 5G」等のキャリア別おすすめ分類が定番化。自社はキャリア別分類ニッチ（mnp_from_*）は整備済みだが、一人暮らし等の主要ニッチでキャリア別導線が未整備。
- **対応：** one_person_roomニッチのLPに「今のスマホキャリアで絞り込む」UI・内部リンク構造を追加。mnp_from_*ニッチへの内部誘導線として機能させる。
- **期待効果：** ユーザーの意思決定コスト低下でCV率向上・内部回遊率改善。

#### 4-3. 光回線乗り換えニッチの「2年サイクル活用」訴求追加（moving_fiber）
- **課題：** 「光回線は2年ごとに乗り換えがお得」という戦略訴求が競合で拡大中。自社のmoving_fiber `primary_appeal` は新規導入寄りで、長期利用者向けの乗り換え誘導訴求が薄い。
- **対応：** moving_fiberのprimary_appealに「2年ごとの乗り換えCB活用例（2年で60,000〜100,000円相当回収・条件あり）」を追加。同時に長期利用者向けの独立サブニッチLP化を検討。
- **期待効果：** 既存ユーザーへの訴求拡大と「乗り換え周期需要」の取り込み。

### 優先度：中

#### 4-4. 副回線ニッチ（second_line）のpovo「180日ルール」具体訴求強化
- **課題：** 格安SIM比較サイトでpovo基本料0円の特殊ポジションが定着。しかし「180日以内にトッピング購入が必要」という管理コストが乗り換えの心理的ハードルになっている。自社LPでは言及があるが、具体的な回避方法（最安390円トッピング・リマインド設定）の記述が薄い。
- **対応：** second_lineニッチのLPに「180日自動解約を防ぐ最安維持方法（月平均65円〜）」を具体的数値で訴求。不安解消型のFAQ形式で記述。
- **期待効果：** povo訴求時のCV率向上（不安解消→申込意欲増加）。

#### 4-5. ポケットWiFi系LPの「後悔・落とし穴」ネガ訴求の見出し化
- **課題：** 競合で「後悔しないためのポケットWiFi選び」「よくある落とし穴」系の見出しが上位取得に成功している。自社はpain_pointsにほぼ同内容が含まれるが、LP見出しに反映されていない可能性がある。
- **対応：** business_traveler・freelancer等のポケットWiFi系ニッチのLPファーストビューに「○○の落とし穴」「後悔しないための確認ポイント3つ」等のネガティブフックを見出しに採用。
- **期待効果：** 不安層・比較検討中間層のクリック率（CTR）向上。

#### 4-6. 格安SIM系LPの電波品質安心訴求の前面化（price_sensitive・mnp_from_docomo等）
- **課題：** 競合各社が「ドコモ回線そのまま」「au回線品質保持」等の電波品質安心訴求を強化中。自社のprice_sensitiveでは言及があるが、ファーストビューより後段に配置されている可能性がある。
- **対応：** 格安SIM系LPのファーストビューに「電波品質はキャリアのまま」を明示するアイコン・バッジ表示を追加。ahamo（ドコモ回線）・LINEMO（SB回線）の回線種別を図解化。
- **期待効果：** 格安SIMへの切り替えを躊躇している「電波不安層」へのCV率向上。

### 優先度：低

#### 4-7. シニア・親世代系LPのUI改善（elderly_simple・senior_utility）
- **課題：** 現行の訴求内容は適切だが、ターゲット（60代以上）に対してフォントサイズ・視認性・CTAボタンサイズが競合比で改善余地がある可能性。
- **対応：** LP改善チームにUX観点でのシニア向けUI確認を依頼（フォント18px以上・CTAボタン大型化・1ページで1アクション設計）。
- **期待効果：** シニアターゲットのCV率改善・離脱率低下。

#### 4-8. ガス代節約ニッチ（gas_saving）のコンテンツ拡充
- **課題：** 競合で「電気代節約」は激戦区だが「ガス代節約」はまだ比較的競合が少ない。自社はgas_savingニッチを保有するが、コンテンツ量・更新頻度が電気系ニッチより少ない可能性。
- **対応：** gas_savingニッチで「プロパンガスの適正価格確認→切替」フローを具体化したコンテンツを追加。競合の少ないブルーオーシャンを先行取得する。
- **期待効果：** SEOロングテールKWの上位取得・CV件数の増加。
