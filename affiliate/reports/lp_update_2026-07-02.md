## 改善ニッチと変更内容

### 1. `one_person_room`（一人暮らし即日WiFi）

**変更理由:**
競合LP分析で「3年実質月額フレーム」「CB差引後月換算」が主流の訴求パターンと判明（research_2026-06-30.md）。自社のLP primary_appeal に情報はあるが、LP冒頭での「月換算比較」が弱く競合に見劣りしていた。

**search_keywords 追加（4件）:**
- `ZEUS WiFi SoftBank Air 3年 総費用 キャッシュバック 差引 実質 比較 2026`
- `SoftBank Air 実質月額 CB差引 3年 月換算 計算 一人暮らし 2026`
- `一人暮らし WiFi 後悔しない 失敗しない 選び方 確認 2026`
- `ZEUS WiFi デメリット 注意点 解約 端末 返却 費用 確認 2026`

**primary_appeal 変更（冒頭にサマリーを追加）:**

変更前:
```
一人暮らし向けホームルーター比較2026：月額を抑えたいなら...
```

変更後:
```
【3年総費用CB差引後の実質比較2026】費用重視→ZEUS WiFi：3年実質約87,548円・月換算約2,432円相当（CB10,000円差引後・初期費用込み・条件あり）／CB最大化→SoftBank Air：3年実質約161,248円目安・月換算約4,479円相当（CB最大35,000円差引後・8ヶ月後受取・条件あり）・いずれも解約金なし・工事不要・翌日発送・一人暮らし向けホームルーター比較2026：...
```

**CV率向上効果の見込み:**
- 比較軸を「月額」から「3年実質月換算」に格上げすることで、価格判断の基準を訪問者に提供
- 「月換算2,432円 vs 4,479円目安」という実数値が、申込前の迷いを解消
- 「後悔しない」「デメリット」等の不安解消KWを追加してCV意欲の高いユーザーを捕捉

### 2. `price_sensitive`（格安スマホ乗換・通信費削減）

**変更理由:**
競合が「使い放題派 vs 低容量節約派」の分岐訴求を導入しており、「用途に合わせた最適解」を提示するLP構成が主流化（research_2026-06-30.md ギャップ3・4参照）。また「混雑時速度」への言及が自社では弱かった。

**pain_point #4 変更:**

変更前:
```
格安SIMに変えると室内・地下・郊外で電波が繋がりにくくなる場所は増えるのか・自宅エリアの電波確認は申込前にできるのか
```

変更後:
```
ahamo（ドコモ回線）・楽天モバイルに乗り換えると自宅・室内・地下・お昼（12〜13時）や夜（20〜23時）の混雑時間帯で速度や電波品質が今より悪くなるのか・乗換前にエリアや速度を確認する方法はあるのか
```

**search_keywords 追加（4件）:**
- `格安SIM 使い放題 無制限 おすすめ ahamo 楽天モバイル 比較 2026`
- `格安SIM 月5GB 以下 安い LIBMO LINEMO 節約 乗換 比較 2026`
- `ahamo ドコモ回線 お昼 夜 混雑 速度 落ちる 確認 実測 2026`
- `格安SIM 乗換 後悔 失敗しない 確認 チェックリスト 2026`

**primary_appeal 変更（冒頭に用途別分岐を追加）:**

変更前:
```
ahamo（月2,970円/20GB・ドコモ回線品質そのまま）へのMNP乗換で...
```

変更後:
```
【用途別おすすめ】▼月20GB以上・使い放題派→ahamo（月2,970円/20GB・ドコモ回線品質そのまま・混雑時も同水準）または楽天モバイル（月3,278円/無制限・解約金なし）▼月5GB以下の節約派→LIBMO（月980円〜/3GB・ドコモ回線）・詳細は以下→ ahamo（月2,970円/20GB・ドコモ回線品質そのまま）へのMNP乗換で...
```

**CV率向上効果の見込み:**
- 「自分はどっち？」の分岐ナビで訪問者が最適案件に早期到達→離脱率低下
- 「混雑時も同水準」の明記でahamo・楽天モバイルの電波品質不安を先制解消
- 混雑時速度不安KWでの流入ユーザーのCV率向上が見込まれる

### 3. `electricity_one_person`（一人暮らしの電気代節約）

**変更理由:**
本日（2026-07-02）時点で電気代補助金は既にスタート済み。前回の primary_appeal は「補助再開まで残りわずか」という訴求で、時制がずれた状態になっていた。「今月申込→8月請求からW節約効果」という正確な緊急訴求に更新。

**search_keywords 追加（3件）:**
- `楽天でんき 7月 申込 8月 補助金 節約 W効果 タイミング 2026`
- `電気代 補助金 7月 スタート 新電力 乗り換え 今すぐ 間に合う 2026`
- `楽天でんき 解約 違約金 なし 乗換 手続き 簡単 確認 2026`

**primary_appeal 変更（冒頭訴求を時制修正＋強化）:**

変更前:
```
【2026年7月補助再開まで残りわずか】今すぐ楽天でんきに切り替えておくと7月の補助金再開月から節約効果と補助金...補助再開前の申込推奨・
```

変更後:
```
【2026年7月 電気代補助スタート！今がW節約チャンス】今すぐ楽天でんきに切り替えると8月請求分から電気代補助金（8月分3.5円/kWh・9月分4.5円/kWh・申請不要・自動適用・政府発表に基づく・時期・金額は変更の可能性あり・新電力も対象）と新電力切替の節約効果を同時に受けられる・7月中に申込完了で8月請求からW節約効果が見込まれる（申込から開通まで通常5〜10日程度・早めの申込推奨）・
```

**CV率向上効果の見込み:**
- 「再開まで残りわずか」→「スタート！」に修正し、時制の誤りを解消
- 「7月申込→8月からW節約」という具体的タイムラインで申込の締切感を演出
- 「違約金なし」「解約簡単」のKWで申込障壁の不安を解消

## 景表法遵守チェック

- ❌ 「最安」「業界No.1」「絶対」「必ず」「最速」「日本一」→ 使用なし
- ✅ CB条件・受取時期は「条件あり」「申込時要確認」を明記
- ✅ 補助金は「政府発表に基づく・時期・金額は変更の可能性あり」を付記
- ✅ 節約額は「見込まれる」「目安」「エリア・使用量により異なる」表現を維持
