## 2. 改善対象ニッチと変更内容

### ニッチ①: `one_person_room`（一人暮らし即日WiFi）
**対応PARTNERED案件**: softbank-air

#### primary_appeal
| | 内容 |
|---|---|
| **変更前** | 工事なしで即日開通、契約縛りなし、月5,000円以内 |
| **変更後** | コンセント挿すだけ・申込翌日から使える・解約金なし・初期費用なしのプランあり |

**改善理由**: 「月5,000円以内」という上限訴求は競合の「月2,000〜3,000円台〜」の下限訴求に負けていた。数字を外し、申込の摩擦を下げる「翌日から使える」「解約金なし」「初期費用なし」の具体的ベネフィットに転換。

#### pain_points
| | 変更前 | 変更後 |
|---|---|---|
| 1 | 明日からネット使いたい、工事は待てない | 申込完了から実際に使えるようになるまで何日かかるのか |
| 2 | スマホのギガがすぐ無くなる | 工事費や初期費用は月額以外にいくら別途かかるのか |
| 3 | 1人だから家用WiFi契約は損な気がする | 賃貸退去のときに解約金や撤去費用は発生するのか |
| 4 | 退去時の解約金が怖い | 端末は購入するのか、レンタルなのか返却しないといけないのか |

**改善理由**: 変更前は「痛み」の描写で止まり、申込直前に生じる具体的疑問を解消していなかった。LP上でこれらの疑問を先回りして回答することで申込離脱を防ぐ。

#### search_keywords
| 変更前 | 変更後 |
|---|---|
| 一人暮らし WiFi おすすめ | 一人暮らし WiFi 月額 いくら |
| 引っ越し ネット 即日 | 引っ越し ネット 即日 工事なし |
| 工事不要 WiFi 1人 | ホームルーター 賃貸 解約金 |
| 賃貸 WiFi すぐ使える | 一人暮らし ホームルーター 比較 2026 |
| 一人暮らし ホームルーター | コンセント WiFi 申込 流れ |
| 新生活 WiFi 安い | 賃貸 ネット すぐ使える 安い |

### ニッチ②: `electricity_one_person`（一人暮らしの電気代節約）
**対応PARTNERED案件**: rakuten-denki

#### primary_appeal
| | 内容 |
|---|---|
| **変更前** | 基本料金0円・縛りなし・解約金なし |
| **変更後** | 申込5分・工事不要・解約金なし・旧電力会社への連絡代行あり・2026年7月の補助再開タイミングで乗り換えがお得 |

**改善理由**: 研究レポートH-2「補助金再開の時事ネタ追加」に対応。「申込5分」「連絡代行あり」で申込摩擦を下げる訴求に変換。競合（エネチェンジ等）が「7月から補助再開」で上位取得中のため時事性を盛り込んだ。

#### pain_points
| | 変更前 | 変更後 |
|---|---|---|
| 1 | 電気代が月4,000-8,000円で高いと感じる | 電力会社を変えると今の会社への解約手続きを自分でしないといけないのか |
| 2 | 新電力が多すぎて選べない | 新電力に変えて電気の供給が止まったり不安定になることはないのか |
| 3 | 解約金や違約金が怖い | 途中で解約するときに違約金は発生するのか |
| 4 | 切替手続きが面倒そう | 申込後に2社から請求書が届いて管理が複雑にならないか |

**改善理由**: 変更前は「高い」「多すぎて選べない」などの状況描写。変更後は電力切替を踏み止めている具体的な不安（手続き煩雑・供給不安・違約金・請求管理）に特化し、LP内で直接解消する設計にした。

|---|---|
| 一人暮らし 電気代 節約 | 電気代 節約 一人暮らし 乗り換え |
| 新電力 一人暮らし おすすめ | 新電力 切り替え 手続き 流れ |

### ニッチ③: `second_line`（副回線・予備用の格安プラン）
**対応PARTNERED案件**: rakuten-mobile / wifi-tokyo-prepaid

#### primary_appeal
| | 内容 |
|---|---|
| **変更前** | 基本料金0円〜・縛りなし・いつでも解約可 |
| **変更後** | 基本料0円から持てる副回線・使わない月は0円のまま・メイン障害時も即切替・縛りなし |

**改善理由**: 研究レポートH-3「副回線0円訴求の強化」に対応。「使わない月は0円のまま」という損失回避の表現を追加し、「メイン障害時も即切替」でユースケースを具体化。povo2.0等の「2026年の新常識」ポジショニングに対抗する。

#### pain_points
| | 変更前 | 変更後 |
|---|---|---|
| 1 | メイン回線が落ちた時の備えが欲しい | 副回線を持つには2台のスマホが必要になるのか、今のスマホで使えるのか |
| 2 | 仕事用とプライベートを分けたい | 使わない月も月額料金は発生するのか |
| 3 | 災害時のリスク分散したい | 180日使わないと自動解約になると聞いたが管理が面倒では |
| 4 | 副回線にお金はかけたくない | メイン回線と同じキャリアから副回線を申し込めるのか |

**改善理由**: 変更前は「なぜ副回線が欲しいか」の欲求描写のみ。申込直前に生じる「デバイスは？」「無駄な月額？」「自動解約？」「同キャリア申込？」という具体疑問に変換し、LPで解消することでCVRを上げる。
